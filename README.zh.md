# React macOS UI

一个 macOS 界面风格的 React 组件库。

Choose your language / 选择语言:
- [English](README.md)
- 中文

## 特性

- 🎨 macOS 风格的 UI 组件
- 🚀 基于 React 和 TypeScript 开发
- 📦 支持按需引入
- 🎯 完整的类型定义
- 💪 更多组件正在开发中...

## 安装

```bash
npm install react-macos-ui
```
## 组件

### Window 窗口

macOS 风格的窗口组件,支持:
- 拖拽移动
- 调整大小
- 最大化/最小化
- 窗口管理

### Dock 程序坞

macOS 风格的 Dock 栏,支持:
- 底部/左侧/右侧停靠
- 自动隐藏
- 图标缩放动画

### AppDisplay 应用图标

用于显示应用图标,支持:
- 自定义图标
- 悬停缩放

## 主题

支持亮色和暗色两种主题模式:
```tsx
// 设置暗色主题
<div data-theme="dark">
<App />
</div>
```
