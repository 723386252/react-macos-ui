## Components

### Window

macOS style window widget, supports:
- Drag and move
- resize
- maximize/minimize
- close
- window manager

### Dock

macOS style Dock bar, support:
- bottom/left/right Dock
- auto-hide
- icon scaling animation

### AppDisplay

display application icons, support:
- custom icon
- hover zoom

## Theme

Supports both light and dark theme modes:
```tsx
// set dark theme
<div data-theme="dark">
<App />
</div>
```
